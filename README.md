# Mon Application Git 

Bienvenue dans mon application version 1.0 !

## Description
Ce projet est une simulation d’un cycle complet de développement avec Git :
- Branche `feature` : ajout d’une fonctionnalité.
- Branche `fix` : correction d’un bug.
- Branche `test` : tests avant la mise en production.
- Branche `develop` : branche d’intégration.

## Fichiers importants
- `app.md` : fonctionnalité principale
- `bugfix.txt` : correction du bug
- `test-report.md` : rapport des tests
- `CHANGELOG.md` : journal des modifications
